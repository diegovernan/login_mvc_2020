# Login - MVC - 2020

## Instalação (localhost)

Clonar o projeto
```bash
$ git clone git@github.com:diegovernan/login_mvc_2020.git
```

Instalar composer
```bash
$ composer install
```

## Configuração

Importar arquivo `login-mvc-2020.sql`

Configurar arquivo `src/Config.php`

## Utilização

Acessar `localhost/login-mvc-2020/public`
